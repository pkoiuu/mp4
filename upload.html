上传视频

浏览器内直接上传,小视频入库,大视频进 Releases

← 返回画廊

第 1 步:配置 GitHub Token

上传需要 Personal Access Token(PAT)。Token 只存在你浏览器的 localStorage,不会上传到服务器。 点此创建 Token ,勾选 repo 权限即可。

第 2 步:选择视频文件

支持 MP4/WebM/MOV 等格式。小视频(≤100MB)上传到 videos/ 目录,大视频(>100MB)上传到 Releases。

拖放视频文件到此处

或 点击选择文件

上传说明

  • 小视频上传后,GitHub Action 会自动扫描并更新画廊(约 1 分钟)
  • 大视频上传到 Releases,需要填写 Release tag(如 v1)
  • Token 只在浏览器本地保存,关闭页面后仍保留,可随时清除
  • 上传中请保持页面打开,关闭页面会中断上传

视频通过 GitHub 仓库托管 · 直链由 jsDelivr CDN 加速 · 大文件走 Releases
